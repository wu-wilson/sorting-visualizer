# **Sorting Visualizer**

This repository contains the code for a sorting algorithm visualizer I made.
Currently, this visualizer contains animations for:

- Bubble Sort
- Selection Sort
- Insertion Sort
- Merge Sort
- Quick Sort

### **Running the Web App**
